# Carrier Loss Report Analyzer

Upload carrier reports (PDF, XLS, XLSX), analyze loss ratio, growth %, and retention %, then download a clean CSV.

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## How to Deploy on Render

1. Push this project to GitHub.
2. Go to [render.com](https://render.com), create a new Web Service.
3. Use `streamlit run app.py` as the start command.
